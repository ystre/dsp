== Quick start

Example codes in documentation can be run with `./scripts/rundoc.sh`.

All code blocks can be dumped via `./scripts/dump-code-blocks.sh`.

Automated testing can be run with `./scripts/run.sh all`.

[source,bash]
----
./scripts/rundoc.sh devenv create-container build-in-container
----
