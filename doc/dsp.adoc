== Quick start

Example codes in documentation can be run with `./scripts/rundoc.sh`.

Automated testing can be run with `./scripts/run.sh`.

[source,bash]
----
./scripts/rundoc.sh devenv create-container build-in-container
./scripts/run.sh build test ft-kafka-producer perf-tcp perf-kafka-producer perf-dsp doc
----
